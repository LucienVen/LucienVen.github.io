如果windows的体验能有macOS的80%..

26/Jun 2019
记录

成为4、5年mac用户之后，终于windows也追上来了。我仍然是赞同mac对于开发者更加合适，但是我也觉得在docker发展与普及后，windows上的开发体验也是越来越来好了。所以，如果是现在的我，平衡性价比的情况下，我确实会选择组织一台台式机，但是还是加上一句——在平衡日常使用的情况下。但是此文，还是甚少评论开发环境之间的差异，因为windows敢把命令行弄好吗？？？hhhh

作为一个15年大二暑假决心吃土一年买了一台macbook air的苹果用户而言，macOS真香，我可是吹了好几年。

“如果windows的体验能有macOS的80%”，为什么突然有这样的想法呢？最开始的来源居然是某老果粉”拨乱反正“，组装了一台差不多上万的windows台式，然后就沉浸在无限的快乐中。那么，现在，这句话能不能成立呢？

15年的电脑到现在19年，特别是后来我使用air外接一个4k（实际使用2k）显示器之后，那种拖滞迟缓感，让我真实体验到——macbook 真贵。

工作一年以来，经历的公司的电脑都是windows系统，一家配了一台i3台式（出差时使用过i5 thinkpad，和i7 thinkpad），而现在准备离开的这家，是i5 + 固态，台式，虽然觉得这个配置是标配，但是还是这基本的操作，我体验到了性能的快乐。

下图日常后台运行软件，4*4的图标为基操，图片还是少了docker，windows下的LNMP集成环境visual NMP等。

苹果电脑有什么好呢？

叛教的老果粉当年安利我的时候，我也正处在人生的某个节点——我将要做什么呢？软件工程专业，可是专业课一塌糊涂，我还要不要做一个程序员呢？为什么配置环境这么麻烦呢？为什么叛教哪个写代码这么香呢？所以，可能我犯了器材党的错误。

但是这个错误，也让我由此迈向搬砖之路 ε=ε=ε=┏(゜ロ゜;)┛，这是我大学期间做的最正确一个选择。

回到标题，苹果电脑有什么好的呢？

  1. macOS
  2. 屏幕

这样就很巧了，macbook air预示着贫穷的我。苹果电脑的系统真的很棒。我继续列一列：

  1. 触摸板
  2. 多桌面
  3. 完善的命令行+简单的环境配置
  4. 玩不了国服LOL

而macOS的好，离不开该生态的发展，按照我个人不太正确的描述：硬件平台给予软件应用开发带来便利，而软件的开发与使用又能反馈给平台，给予系统生态的提升，从而又能促进硬件平台的发展（其中一部分的意思也就是说推广与盈利）。

另外对于开发者而已，”完善的命令行+简单的环境配置“，这个是重中之重吧，macOS 是 类Unix系统，Linux也是类Unix系统鸭，命令行与桌面应用的完美结合，没错就是macOS。

windows 10 开始香了？

用户体验这点mac真的很棒。这里的很棒，指的是1.硬件基础的好，2.软件使用带来的好。硬件而言，我感觉到最明显的就是触摸板，让使用者可以离开鼠标，也就减少了使用途中右手从键盘到鼠标的转移距离。软件而言，mac下的quicklook，aflred，Unclutter等软件，总能让人感到便捷与实用。

我所描述的windows的香，也是主要针对windows下的日常使用体验，或者说软件的进步使得设备使用体验的提升。

相信诸君也觉得mac真的很贵吧！以最为常见的MacBook pro为例，

  1. 2.4GHz 四核第八代 Intel Core i5 处理器
  2. Intel Iris Plus Graphics 655 图形处理器
  3. 256GB 固态硬盘

我确实在硬件方面不太懂，但是我也能感觉到，这样的硬件配置，与13000+的价格，what always happens, life?

在windows上，我为什么觉得能达到macOS的香呢？

这段我打算记录下我现在台式机上装的软件。前面也说到，本文想描述的是软件的进步使得设备使用体验的提升。

火绒安全

这个必须放第一位吧！！！

国内厂商最恶心人的地方，不就是在于无休止的广告，无休止的弹窗吗！！！真实过分！

所以，火绒安全的”弹窗拦截“是人性之光吧！不止内置一些拦截规则，还能自定义添加拦截的窗口，实话说，90%以上的弹窗鸭广告鸭，都能干掉了。

配套来说，chrome浏览器的广告拦截插件也应该提上一嘴的，但是我选择把这个放在后面再提。

Steam WALLPAPER ENGINE 壁纸引擎

谁能想到动态壁纸会真香？

谁能想到着动态壁纸应用时steam的，好端端的游戏不卖卖壁纸？emmm 真香。

刚刚下载的那段时间，每天的娱乐/摸鱼活动就是，找壁纸——而且，你电脑壁纸那个结衣老婆会动啦（笑

Stardock Fences3 桌面图片整理

我是个喜欢简洁的人。同时我又常常是一个失败的整理者。我的windows桌面是被Stardock Fences3拯救的。

能把windows桌面上各种奇奇怪怪的文件、快捷方式、应用等等图标隐藏起来，却又不影响使用的便捷，甚至还能按照个人的想法分好区，多棒鸭！再者加上上面的WALLPAPER ENGINE，xixixixi…

Wox——Windows 上的 Alfred

Wox是windows上的效率启动器，可以想象为mac上的小黑帽Alfred，是时候离开鼠标了吧！

少数派介绍文章 - Windows 上的 Alfred，免费开源的效率启动器：Wox

QuickLook——空格预览文件

既然都能把mac上的Alfred搬回来了，那么，空格预览文件功能也不能落下吧。预览markdown文件时候还能带有格式，我也忘了mac上预览md文件，是不是已经渲染好了…

CCleaner PC的清理软件

某天我找我哥玩的时候，看到那个360安全管家（PC上最大的流氓）就浑身难受，还有铺天盖地的弹窗广告，然后我照例案例火绒安全的时候，他说360还挺好用的，需要用到360的垃圾清理……..

emmmmmmmmmmmm……….

相信我，火绒安全+CCleaner，然后放弃你的360全家桶。

ShadowsocksR / Shadowsocks 梯子

这个不多说了。

google chrome 浏览器

chrome浏览器，如果用起来觉得一般般，那么肯定是你没有装插件的问题。

为什么你没有装插件呢？我觉得可能是你打不开chrome 网上应用店； 为什么你打不开呢？我觉得可能是你没有搭梯子（笑，其实也有很多离线包下载的

chrome的牛逼之处，就是插件了，如果你没有使用插件，我觉得你也未算真正使用好这个浏览器，什么很多插件做着做着，都变成PC APP了，如postman。当然，火狐浏览器firefox，也有应用店啊，其实都差不多的，不同生态而已。chrome上有的，firefox应该也有。其实浏览器的选择还是看个人喜欢，个人爱好，个人习惯。

但是在这里还是提上浏览器一嘴的原因，是adblock plus，或者其他浏览器网页广告拦截插件，让你的页面干干净净，不被广告打扰，很重要。其实我觉得这点，不怎么需要提起，但是我就是不明白为什么这么多人能忍受广告的呢？

同步类应用

Dropbox

需要番羽土蔷，这算是不是甚好的地方。免费5G空间，假设不同步图片的情况下，同步保存文件与代码，还是很不错的。就不会出现家里和公司，每次都需要拿个u盘把东西拉来拉去。

OneDrive

微软家的，微软最近还真是8错鸭！这是我最近常用的。用途与Dropbox有重叠，对比而已明显的优势在于不需要番羽土蔷。同样免费5G空间，另一方面可能就是微软自家产品，在自家环境下可能更好用吧，优化得更好。

有道云笔记

笔记同步。支持全平台吧。里面可以创建文件夹之类，进行笔记管理还是不错的。虽然你也可以使用Dropbox / OneDrive，同步自己的word文档，md文档之类，但是还是能找到适用场景的。但是想吐槽一下，为什么不支持markdown呢？

如果有人问为什么不用印象笔记，我的回答：有丶穷orz

TIM——纯净版QQ

我认为微信属于更私人的领域，对于工作，我更倾向于使用QQ，因为我日常都不用。可以把工作与日常分隔开。但是具体还是要看贵司的要求了。

但是QQ？笑了。冗肿。当年成为macOS用户之后，mac 版本的 QQ真是干净啊！回忆中PC版QQ也令人太难受了吧。

但是现在有TIM啊，可以理解为纯净版的QQ，专注于办公，而且接入微云，文件也是可以存储在微云上的。

cmder——命令行工具

对于开发者而已，命令行还是必备的吧。

描述cmder，引用一句：其虽没oh-my-zsh那么逆天的存在，却也甚是好用，至少要比Windows原生Cmd好出了天际。

简单列一下优点

  1. 快捷键唤起
  2. 自定义样式
  3. 支持bash命令
  4. 带有ssh
