Hugo基本使用

17/Jan 2019
hugo 记录

hugo基本使用记录，以免自己忘记了（笑

本站使用的hugo主题是hugo-redlounge，原因是可配置程度稍高，能设置更多的个人信息，现阶段而言这个主题比较适合我。但是我最喜欢的主题应该是Angel’s Ladder，Blog是记录内容的地方，而这个主题十分简约，我很喜欢。

另外，hugo-redlounge的表格和引用样式实在太蠢了hhhhh

关于hugo调试

$ hugo server --theme=hugo-redlounge --buildDrafts

开始调试模式，可以在浏览器输入localhost:1313，查看页面变化

$ hugo --theme=hugo-redlounge --baseUrl="https://lucienven.github.io/"

编译网站内容，并把静态页面输出到/public目录下，然后我们可以通过git操作把/public目录下的全部内容上传到github，然后我们就可以通过上面命令中的baseUrl访问了。

创建文章

$ hugo new about.md

about.md 自动生成到了 content/about.md

$ hugo new post/first.md

则文章创建到post目录下

文章格式设置

关于文章内容

Key Remark
draft 是否编译（true: 不再编译 / false: 编译）
Tags 标签
Description 页面描述（Html titile）
Categories 分类
< !–more–> 在more前面的内容，会被当做文章摘要；若无改设置，则自动获取文章前80字符作为摘要。

查看内容

  • 查看标签Tags
    • 地址栏后面添加 /tags
    • eg: https://lucienven.github.io/tags/
  • 查看分类Categories
    • 地址栏后面添加 /categories
    • eg: https://lucienven.github.io/categories/

一些事项：

  1. 文章命名，开头若果不是英文，则会出现错误。
